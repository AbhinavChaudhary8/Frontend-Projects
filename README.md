1. TaskFlow — Task Manager
A productivity app with user authentication, task management, and priority tracking.

**Features:**
- Login & Register system with localStorage
- Add tasks with High / Medium / Low priority
- Filter by All, Today, Pending, Completed
- Live stats dashboard
- Data persists between sessions

---

2. Spendr — Expense Tracker
A personal finance tracker with charts and monthly analysis.

**Features:**
- Add income & expenses with categories
- Donut chart — spending by category
- Bar chart — last 6 months overview
- Navigate between months
- Category-wise breakdown with progress bars

---

## Tech Stack
HTML · CSS · JavaScript · Chart.js · localStorage
